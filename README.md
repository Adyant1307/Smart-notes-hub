A full-stack note management app where users can upload notes, store files in MinIO, search instantly using Elasticsearch, and interact via a React + FastAPI setup.

Features

1 Upload notes (title, tags, file)

2 Search notes (Elasticsearch)

3 File storage via MinIO

4 PostgreSQL + SQLAlchemy

5 Upvote system

6 Tech Stack

FastAPI, React, MinIO, Elasticsearch, PostgreSQL, SQLAlchemy
